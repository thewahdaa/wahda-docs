# The Wahda Cloud — Documentation

The documentation for [The Wahda Cloud](https://thewahda.com) — pay-as-you-go cloud infrastructure in India.

Live site: **[docs.thewahda.com](https://docs.thewahda.com)**

## Local development

```bash
npm install
npm start         # dev server on http://localhost:3000
npm run build     # static build into ./build
```

## Capture toolkit

Headless Skyline screenshots + GIFs are generated by Playwright scripts under `capture/`. See `capture/README.md` for details.

## Stack

- [Docusaurus 3](https://docusaurus.io/) (static site generator)
- TypeScript + Mermaid diagrams
- [Geist](https://vercel.com/font) font, slate palette, primary `#2563eb` (matches thewahda.com)
- Deployed to GitHub Pages with CNAME → `docs.thewahda.com`

## Contributing

Open a PR against `main`. The `deploy.yml` workflow auto-publishes after merge.
